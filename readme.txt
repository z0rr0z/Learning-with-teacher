Построение моделей для прогнозирования покупательской активности

Заказчик исследования: интернет-магазин «В один клик».
Цель исследования: построить модели для прогнозирования активности клиентов и подготовка предложений для удерживания активности постоянных клиентов.

Решены следующие задачи:
- предобработка данных;
- исследовательский анализ данных;
- корреляционный анализ признаков;
- построение моделей прогнозирования активности клиентов;
- выделен отдельный сегмент клиентов для углубленного анализа и подготовить предложения по работе с этим сегментом для увеличения покупательской активности.

Рассмотренные модели: KNeighborsClassifier, DecisionTreeClassifier, LogisticRegression, SVC.
Лучшей моделью является логистическая регрессия с l2-регуляризацией и "обратной силой" 3, с использованием стандартизации и 9 числовых факторов из 10 исходных.
